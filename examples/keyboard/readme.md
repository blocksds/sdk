# BlocksDS SDK examples: Keyboard

This folder contains examples of how to use the virtual keyboard of libnds.

- `custom_keyboard`: Loads a custom keyboard with libnds.
- `default_keyboard`: Loads the keyboard of libnds and reads its state manually.
- `stdin_read_blocking`: Uses `read()` to read `stdin` in a non-blocking way.
- `stdin_read_nonblocking`: Uses `read()` to read `stdin` in a blocking way.
- `stdin_scanf`: Uses `scanf()` to ask the user for input with libnds' keyboard.
